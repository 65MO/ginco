Édition d'un Tableau de concepts

ÉcranÉditer un Tableau de concepts⚓

RéglementaireAfficher le Tableau de concepts⚓

Pour afficher un Tableau de concepts dans la zone de travail :

  • Double-cliquez sur le nom du Tableau de concepts dans l'explorateur,

    ou,

  • Sélectionnez le Tableau de concepts dans l'explorateur puis cliquez sur Afficher la sélection.

AttentionImportant⚓

Les champs suivis d'un astérisque rouge sont obligatoires.

  • 1 Titre

    Saisissez le titre du Tableau de concepts.

    RemarqueRemarque⚓

    Ce titre sert à identifier le tableau dans l'explorateur de vocabulaires et dans les exports texte hiérarchique.

  • 2 Langue

    Sélectionnez dans la liste déroulante la langue du Tableau de concepts.

    AttentionImportant⚓

    • Un Tableau de concepts ne peut avoir qu'une seule langue.

    • La liste des langues dépend du paramétrage de votre vocabulaire.

  • 3 Trier les concepts par ordre alphabétique

    1. Lorsque l'option est cochée, les concepts du tableau sont exportés dans l'ordre alphabétique lors d'un export texte hiérarchique.

    2. Lorsque l'option n'est pas cochée, les concepts du tableau sont exportés dans leur ordre d'ajout au tableau lors d'un export texte hiérarchique.

      Remarque : L'ordre d'ajout des concepts est visible dans la colonne Ordre de la liste des concepts.

  • 4 Tableau parent

    Cette rubrique permet de désigner un autre Tableau de concepts comme parent du tableau actif afin de créer une relation hiérarchique entre les 2 tableaux et ainsi d'imbriquer ces 2 tableaux dans les exports texte hiérarchique.

    RéglementaireÉditer le tableau parent⚓

    • Cliquez sur le bouton Sélectionner un tableau parent pour désigner un tableau comme parent du tableau actif.

    • Cliquez sur le bouton Supprimer le tableau parent pour supprimer le tableau désigné comme parent du tableau actif.

  • 5 Concept parent

    Cette rubrique permet de désigner un Concept de l'arborescence des concepts comme parent du tableau actif afin que ce tableau soit exporté sous ce Concept dans les exports texte hiérarchique (et non à la racine du vocabulaire).

    RemarqueRemarque⚓

    Lorsqu'un Concept parent est désigné, seuls les enfants de ce concept sont proposés dans la fenêtre de sélection des concepts du tableau.

    RéglementaireÉditer le concept parent⚓

    • Cliquez sur le bouton Sélectionner un concept parent pour désigner un concept comme parent du tableau actif.

    • Cliquez sur le bouton Supprimer le concept parent pour supprimer le concept désigné comme parent du tableau actif.

  • 6 Concepts

    Liste des Concepts qui composent le tableau.

    RéglementaireÉditer la liste des concepts⚓

    • Cliquez sur le bouton Ajouter un concept pour ajouter un Concept à la liste :

      • Si vous n'avez pas désigné de Concept parent, Ginco propose tous les termes de premier niveau (TT[1]) comme membres possibles du tableau.

      • Si vous avez désigné un Concept parent, Ginco propose les enfants de ce concept comme enfants possibles du tableau.

      Remarque : Ginco ne propose pas les Concepts orphelins[2] ; il ne propose que des concepts frères (termes de tête ou enfants d'un concept).

    • Cliquez sur le bouton à droite de la ligne d'un concept pour le supprimer de la liste.

    • Double-cliquez sur la ligne d'un concept pour ouvrir l'onglet correspondant à ce Concept dans la zone d'édition.

Références

  1. Terme de Tête
    Top Term (anglais) TT

    Un Terme de Tête est un Terme préférentiel d'un Concept qui n'a pas de concept générique dans le vocabulaire : il se situe donc à la racine du vocabulaire. Il constitue un point d'entrée dans une branche d'un thésaurus.

  2. Concept orphelin

    Un Concept orphelin est un concept qui n'est pas rattaché à un concept parent dans l'arborescence des concepts et qui n'est pas déclaré comme Terme de tête. Il peut avoir des descendants (branche du thésaurus en phase d'élaboration).
