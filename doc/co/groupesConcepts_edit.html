Édition d'un Groupe de concepts

ÉcranÉditer un Groupe (de concepts ou dynamique)⚓

RéglementaireAfficher le Groupe⚓

Pour afficher un Groupe dans la zone de travail :

  • Double-cliquez sur le nom du Groupe dans l'explorateur,

    ou,

  • Sélectionnez le Groupe dans l'explorateur puis cliquez sur Afficher la sélection.

AttentionImportant⚓

Les champs suivis d'un astérisque rouge sont obligatoires.

123456
  • 1 Titre
  • 2 Type
  • 3 Langue
  • 4 Notation
  • 5 Groupe parent
  • 6 Concepts
  • 1 Titre

    Saisissez un titre décrivant le regroupement.

    RemarqueRemarque⚓

    Ce titre sert à identifier le groupe dans l'explorateur de vocabulaires et dans les exports SKOS.

  • 2 Type

    Sélectionnez dans la liste déroulante le type du groupe (Thématique, Facette, Domaine ou Microthésaurus).

    RemarqueRemarque⚓

    La liste des valeurs est paramétrable par l'administrateur de l'application, en fonction des besoins.

  • 3 Langue

    Sélectionnez la langue dans laquelle est exprimé le titre du Groupe.

    AttentionImportant⚓

    La liste des langues disponibles dépend du paramétrage de votre vocabulaire.

  • 4 Notation

    Saisissez une notation symbolique pour le groupe.

  • 5 Groupe parent

    Cette rubrique permet de désigner un autre Groupe comme parent du groupe actif afin de créer une relation hiérarchique entre les deux groupes.

    RéglementaireÉditer le groupe parent⚓

    • Cliquez sur le bouton Sélectionner un groupe parent pour désigner un groupe comme parent du groupe actif.

    • Cliquez sur le bouton Supprimer le groupe parent pour supprimer le groupe désigné comme parent du groupe actif.

  • 6 Concepts

    Liste des Concepts ajoutés au Groupe de concepts.

    RéglementaireÉditer la liste des concepts⚓

    • Cliquez sur le bouton Ajouter un concept pour ajouter un Concept à la liste.

    • Cliquez sur le bouton à droite de la ligne d'un concept pour le supprimer de la liste.

    • Double-cliquez sur la ligne d'un concept pour ouvrir l'onglet correspondant à ce Concept dans la zone d'édition.

    AttentionParticularité du Groupe dynamique⚓

    Dans le cas d'un Groupe dynamique, il suffit d'ajouter un seul Concept, dit Tête de groupe. C'est l'ensemble des concepts enfants du concept tête de groupe qui seront exportés pour ce Groupe lors des exports SKOS.
